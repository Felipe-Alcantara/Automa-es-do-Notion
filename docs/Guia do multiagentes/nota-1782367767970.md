## Travas
- **`docs/README.md` é arquivo compartilhado e tem índice acumulando entradas de vários
  agentes** (AGENTES + INFRA + CONTRATOS) ainda NÃO commitadas. O Agente 0 deixou sua linha
  de CONTRATOS no working tree mas **não commitou o README** para não varrer o `INFRA.md`
  (Infra) nem o `AGENTES.md` junto. → **Precisa de 1 commit de índice coordenado** (quem
  fechar primeiro commita README + AGENTES.md + INFRA.md + CONTRATOS.md numa unidade `docs:`).
- **Git guide do padrão**: commitar **direto no `main`** (sem branch, salvo feature
  grande/refatoração ampla/alto risco), Conventional Commits, **stage só os arquivos do seu
  escopo** (não usar `git add -A` — há trabalho de 3 agentes no working tree), doc viva no
  mesmo commit, sem segredo. Fonte: `Felipe-Alcantara/Felixo-System-Design` →
  `docs/GIT-POLITICA-DE-VERSIONAMENTO.md` (a pasta local do padrão está **vazia**).
- **Ordem de ondas**: Onda 1 (Agente 0) ✅ fechada → Onda 2 (Infra · Backend · IA) liberada.
